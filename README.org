#+TITLE: ox-html-timeline.el

* Quick start

1. Load the =ox-html-timeline.el=
2. Go to the =example/=
3. Hit =C-c C-e n N= to export =index.org=
4. Navigate to =index.html=
5. Hit =C-c C-v= to preview the html output
